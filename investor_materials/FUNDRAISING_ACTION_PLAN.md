## Week 7-8: Close First Cheques

### Momentum Tactics
- [ ] Once you have 1 committed angel, tell others
- [ ] "We're 40% committed" creates urgency
- [ ] Set a "round closing" date 2-3 weeks out
- [ ] Share milestone updates (waitlist growth, etc.)

### Closing Checklist
- [ ] SEIS Advance Assurance received
- [ ] SAFE template reviewed by lawyer (£500)
- [ ] Banking setup (Tide, Mercury for startups)
- [ ] Cap table tracking (Carta, Vestd)
- [ ] Define first B2B pilot target list (3 design partners)

## Red Flags to Avoid

1. **Don't ghost investors** — always close the loop
2. **Don't oversell traction** — VCs talk to each other
3. **Don't give free equity for "advisor" titles**
4. **Don't accept bad terms** out of desperation
5. **Don't stop building** — shipping = momentum
