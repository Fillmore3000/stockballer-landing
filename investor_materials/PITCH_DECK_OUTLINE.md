## Slide 5: How Pricing Works

### Performance-Linked Valuation

```
Price = IPO (£100) + (Net Yield × Age Multiplier × Form Multiplier)
```

**Positive Events (increase price):**
| Action | Yield |
|--------|-------|
| Goal scored | +£0.25 |
| Assist | +£0.12 |
| Match played | +£0.02 |
| Clean sheet | +£0.10 |

**Negative Events (decrease price):**
| Action | Penalty |
|--------|---------|
| Yellow card | -£0.15 |
| Red card | -£1.50 |
| Penalty missed | -£0.50 |

**Result:** Prices reflect real-world performance, not speculation
