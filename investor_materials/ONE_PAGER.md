# StockBaller — One-Pager

## The Stock Market for Professional Football

**Website:** stockballer.app | **Raising:** £250K Pre-Seed | **Stage:** MVP on Base Sepolia (mainnet prep)

---

## Problem
Sports fans are the most knowledgeable "investors" — but have no way to financially benefit from that knowledge. Betting is gambling (house wins). Fantasy has no real stakes. NFTs are collectibles, not liquid investments.

---

## Solution
**StockBaller** lets users buy and sell tokenized shares of professional footballers. Prices move based on real match performance — goals, assists, clean sheets (up) and red cards, missed penalties (down).

| Feature | Description |
|---------|-------------|
| **Mobile-First** | iOS, Android, Web app |
| **Blockchain** | Base L2 (Coinbase), ERC-1155 tokens |
| **Stablecoin** | USDC settlement, no crypto volatility |
| **Real Data** | API-Football performance oracle |
